# Batch: Artifact System

## Task 1: Artifact system (full build)
Create the complete `src/artifacts/` module per the design spec at `docs/superpowers/specs/2026-07-04-artifact-system-design.md`. Create all files: types, parser, resolver, builder, CLI entry point, runtime loader, and tests. Also wire the test into RunTests.luau and update .gitignore.

### Files to create:

**src/artifacts/types.luau** — shared types:
- `PropertyMetadata = { name: string, codecId: number, codecParams: {number}?, default: any? }`
- `ClassMetadata = { id: number, name: string, properties: {PropertyMetadata} }`
- `ArtifactData = { version: number, classes: {ClassMetadata}, classByName: {[string]: ClassMetadata}, classById: {[number]: ClassMetadata} }`

**src/artifacts/build/parseDump.luau** — API dump JSON parser:
- Input: Roblox API dump JSON string (the standard format with `Classes` array, each class has `Name`, `Superclass`, `Members`)
- For each class member: only `MemberType == "Property"`, only `Serialization.CanSave == true` (treat absent as saveable), skip `notScriptable` tags, skip `BinaryString`, skip `Instance` value types
- Walk superclass chain to collect inherited properties. Ancestor properties first (top of hierarchy), then own properties. Within each group, alphabetical by name.
- Output: `{ClassDef}` where `ClassDef = { name: string, superclass: string?, properties: {PropertyDef} }` and `PropertyDef = { name: string, valueType: string, valueTypeParams: {string}?, hasDefault: boolean, defaultRaw: string? }`
- ValueType parsing: if ValueType is a string, that's the type name. If it's a table `{Name = "Enum", ...}`, extract the enum name. If `{Name = "Optional", ...}` with inner type, mark as optional. Parse `ValueType.Category` / `ValueType.Name` from the standard dump format.

**src/artifacts/build/resolveType.luau** — Roblox type → codec ID mapping:
- Exhaustive mapping: float→0(f32), double→1(f64), int→1(f64), bool→2, string→3, CFrame→6, Color3→7, BrickColor→8, ColorSequence→9, NumberRange→11, NumberSequence→12, UDim→14, UDim2→15, Vector2→16, Vector3→17, Rect→18, Axes→19, Faces→20, PhysicalProperties→21, Font→22, Content→23, UniqueId→24, Enum→25(enumItem)
- Array/{T} → factory 4 with inner type codec ID as param
- Optional<T> → factory 5 with inner type codec ID as param
- Unresolvable → return nil (caller handles skip+warn)
- No warnings printed here — caller (buildArtifact) does that

**src/artifacts/build/buildArtifact.luau** — assembler + binary writer:
- Takes ClassDef[] + a resolveType function
- Assigns sequential class IDs starting from 1 (in alphabetical class name order)
- For each property, calls resolveType. Skips props with nil resolution (prints: `"WARNING: {ClassName}.{PropertyName} ({TypeName}) — no codec, skipping"`)
- Default value parsing: parse raw default strings to native Luau values using @lune/roblox constructors (Vector3.new, CFrame.fromMatrix, Color3.fromRGB, BrickColor.new, UDim2.new, etc. — see spec table). Bool: "true"/"false". Numbers: tonumber. Strings: strip quotes if present.
- Serialize each default through its codec (codec.write → buffer bytes)
- Never-elide list: Name, UniqueId (store empty default bytes for these)
- Binary format: magic "LATA" (4 bytes: 0x4C,0x41,0x54,0x41), u16 version=1, varint classCount. Per class: varint classId, string className, varint propCount. Per property: string propName, varint codecId, varint paramCount, varint* params, varint defaultLen, bytes default.
- Use src/buffer/Writer for binary output
- Returns buffer

**src/artifacts/build/Run.luau** — CLI entry point:
- Usage: `lune run src/artifacts/build/Run.luau [dumpPath] [outPath]`
- Defaults: dumpPath = "data/api-dump.json", outPath = "data/artifact.bin"
- Reads JSON file, calls parseDump, resolveType, buildArtifact, writes to file
- Prints summary: class count, property count, skipped type warnings

**src/artifacts/Artifact.luau** — runtime loader:
- `load(artifactBuffer: buffer): ArtifactData`
- Reads binary format: validates magic "LATA", checks version=1
- Uses src/buffer/Reader
- Looks up codecs from src/serde/init (codecIdToModule, codecIdToFactory)
- For factory codecs: reads params, resolves inner codecs, calls factory(innerCodec)
- For each default: if defaultLen > 0, creates Reader over default bytes, runs codec.read to get native Luau value
- Builds classByName and classById index tables
- Returns ArtifactData

**tests/Artifact.spec.luau** — unit tests:
- Use @packages/frktest, @lune/roblox (if needed for defaults)
- Test resolveType: each leaf type returns correct ID, Array/{T} returns factory, Optional returns factory, unknown returns nil
- Test parseDump: hand-crafted mini JSON with 2 classes, inheritance chain, notScriptable property (skipped), CanSave=false (skipped), BinaryString (skipped)
- Test Artifact round-trip: build small ClassMetadata[] by hand (1 class, 2-3 properties with known defaults), write to buffer via buildArtifact, load via Artifact.load, assert loaded data matches
